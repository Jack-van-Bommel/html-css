Vraag 1: Waarom noem je de eerste HTML-pagina; index.html?

Omdat dat nodig is voor de host om te herkennen dat het in HTML is geschreven.

Met welke HTML versie werken we nu?

Op dit moment werken wij met HTML5.

Met welke CSS versie werken we nu?

Wij werken op dit moment met CSS 3.

Hoe kan het dat een website er net iets anders uit ziet in een andere browser?

Verschillende browsers interpreteren HTML en CSS op andere manieren, daarom kan het resultaat er anders uit komen te zien.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

De meest populair browser is Google Chrome, deze browser wordt door 60% van de mensen die op het internet zijn gebruikt.

Ik heb 16gb aan RAM beschikbaar, 1 google tab gebruikt rond de 300mb. Voor mij zou dat dus 1,875% zijn. Dit verschilt per hoeveel RAM iemand heeft.
