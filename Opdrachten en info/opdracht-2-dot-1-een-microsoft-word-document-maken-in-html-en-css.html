Waarschijnlijk zijn veel tags die tijdens de les zijn besproken je bekend voorgekomen.

Open maar eens Microsoft Word en kijk maar waar je de Heading (<h1> t/m <h3>) kan aanpassen. Ook kan je plaatjes toevoegen in een Word Document en zelfs verwijzingen (<a> tag) maken naar andere documenten/web-pagina’s.

Om dat zelf te ondervinden ga je een klein ‘opstel’ schrijven in HTML en CSS met als onderwerp jouw hobby.

Als het goed is heb je tijdens de les een folder aangemaakt genaamd  'p1-html-css' folder' maak daarin een nieuwe map aan genaamd 'opdracht_2.1', hierin plaats je al je bestanden van deze opdracht.

Eisen:

  • Je hebt 1 HTML bestand

  • Je hebt 1 CSS bestand die wordt ingeladen in je HTML pagina

  • Je laadt minimaal 2 afbeeldingen in.

  • Je gebruikt minimaal 1 <a> tag om naar een externe pagina te verwijzen, denk bijvoorbeeld aan:

    • De Instagram pagina van je voetbalclub
    • Een link naar een YouTube video van een tof moment in e-sport
    • Een link naar een recept wat je hebt gemaakt

  • Je hebt minimaal 2 tags aangepast met CSS. Bijvoorbeeld:

    • De kleur van een <h1> tag aanpassen
    • De kleur van een <p> tag aanpassen
    • Als extra uitdaging kan je ook de grootte van je tekst wijzigen.

  • Een klasgenoot heeft je website gecontroleerd of deze voldoet aan alle bovenstaande eisen. Je zet in je HTML dan ook het volgende: 

    • Deze pagina is gemaakt door: Jouw naam
    • Deze pagina is nagekeken door: Naam van je klasgenoot

Oplevering:

Lever je html en css bestand in.
